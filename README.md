## What it is

MarketMind lets you type a question like *"Is NVIDIA overvalued right now?"* or *"Summarise the last week of news for ASML"* and get a concise, sourced answer generated by Claude. Every response is grounded in data fetched at query time — live price history, fundamentals, and news headlines — so the model never invents numbers it was not given.

It uses only free, open data sources: Yahoo Finance via `yfinance` for OHLCV and fundamentals, and Google Finance RSS via `feedparser` for recent headlines. No Bloomberg terminal, no broker account, no paid API keys beyond Anthropic.

## Quickstart

```bash
git clone https://github.com/RitikPatill/marketmind-llm-finance.git
cd marketmind-llm-finance
pip install -e .
pip install -r requirements.txt
cp .env.example .env
# Edit .env and set ANTHROPIC_API_KEY=sk-ant-...

# Start the API server
uvicorn marketmind.api:app --reload &

# Launch the Streamlit UI
streamlit run app.py
```

Python 3.11 or later is required.

## Usage

**Streamlit UI** — open `http://localhost:8501` in your browser. Enter a ticker in the sidebar (e.g. `ASML`), type a question in the text area, and click Analyse. The answer streams in on the right alongside a Plotly candlestick chart of the last 30 days. Three preset example questions are available as one-click buttons.

**CLI** — stream an answer directly to your terminal:

```bash
python -m marketmind.cli ASML "What are the key risks right now?"
```

**API** — call the backend directly:

```bash
# Raw snapshot (price history, fundamentals, news)
curl http://localhost:8000/snapshot/AAPL

# Streaming LLM answer (Server-Sent Events)
curl -N -X POST http://localhost:8000/query \
     -H "Content-Type: application/json" \
     -d '{"ticker":"AAPL","question":"Is it overvalued?"}'
```

## Architecture

```
┌──────────────────────────────────────────────┐
│             Streamlit UI  (app.py)            │
│  ticker input · question box · stream panel  │
└───────────────────┬──────────────────────────┘
                    │ HTTP POST /query (SSE)
┌───────────────────▼──────────────────────────┐
│          FastAPI backend  (api.py)            │
│  GET /snapshot/{ticker}  POST /query         │
└──────────┬────────────────────┬──────────────┘
           │                    │
┌──────────▼────────┐  ┌────────▼─────────────┐
│  Data layer        │  │  LLM layer            │
│  data.py           │  │  analyst.py           │
│  yfinance (OHLCV,  │  │  build_context()      │
│  fundamentals)     │  │  ask() — streaming    │
│  feedparser (RSS)  │  │  Anthropic SDK        │
└───────────────────┘  └──────────────────────┘
```

Data and news are fetched at query time, serialised into a compact Markdown context block (< 1 500 tokens), and sent to Claude with a system prompt that requires citing specific numbers from the context. This makes hallucination straightforward to detect — every figure in the answer can be verified against `GET /snapshot/{ticker}`.
